1. Introduction

A Binary Search Tree (BST) is a hierarchical data structure in which each node contains a unique key and follows an ordering property.

  • Left subtree contains values smaller than the node.
  • Right subtree contains values greater than the node.
  • Duplicate values are generally not allowed.

Search


function search(node, key):
    if node is null:
        return null

    if key == node.value:
        return node

    if key < node.value:
        return search(node.left, key)
    else:
        return search(node.right, key)
